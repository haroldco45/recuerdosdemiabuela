# 🌿 Recuerdos de mi Abuela — PWA de Pedidos

Aplicación web progresiva (PWA) para pedidos de productos fermentados artesanales naturales.

## Productos disponibles
- **Kéfir de Leche Natural de Vaca** — $35.000/litro
- **Kéfir de Agua** — $25.000/litro
- **Kombucha Artesanal** — $35.000/litro

## Características
- ✅ Instalable como app (PWA) en Android e iOS
- ✅ Funciona offline (Service Worker)
- ✅ Hora real de Colombia (UTC-5)
- ✅ GPS para compartir ubicación
- ✅ Calculadora de pedido en tiempo real
- ✅ Envío de factura por WhatsApp al 300 163 8585
- ✅ Acordeón de beneficios por producto
- ✅ Colores y estilo artesanal/natural
